ir_b
====

irb anywhere by calling 'ir b' (need a whitespace!).

Install
----

    gem install ir_b

Usage
----

    ir b

or

    ir-b

Detail
----

* the method 'ir' is Kernel#ir and start irb session with specified binding
* the method 'b' is an alias of binding
* the method IrB.- start irb session
